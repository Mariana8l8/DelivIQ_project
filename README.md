Інструкція з запуску проєкту

1. Необхідне програмне забезпечення
Перед початком роботи на комп’ютері має бути встановлено:
  •	Visual Studio 2022 (версія 17.x або новіша)
  •	Робоче навантаження ASP.NET and web development
  •	.NET SDK 8.0
  •	Microsoft SQL Server (LocalDB або повна версія)
  •	Git (для клонування репозиторія)

2. Клонування репозиторія
  1.	Відкрити командний рядок або Git Bash
  2.	Перейти до папки, де буде збережено проєкт
  3.	Виконати команду:
git clone <URL_репозиторія>
  4.	Після завершення клонування перейти в папку проєкту
	
3. Відкриття проєкту у Visual Studio
  1.	Запустити Visual Studio
  2.	Обрати Open a project or solution
  3.	Відкрити файл рішення з розширенням .sln
  4.	Дочекатися завершення завантаження залежностей
    
4. Налаштування підключення до бази даних
  1.	Відкрити файл appsettings.json
  2.	Перевірити або змінити рядок підключення:
"ConnectionStrings": {
"DefaultConnection": "Server=(localdb)\\MSSQLLocalDB;Database=DelivIQDb;Trusted_Connection=True;"
}
  3.	Переконатися, що SQL Server доступний на локальному комп’ютері

5. Застосування міграцій бази даних
  1.	У Visual Studio відкрити Tools → NuGet Package Manager → Package Manager Console
  2.	Обрати проєкт з DbContext
  3.	Виконати команду:
Update-Database
Після виконання команди база даних буде створена автоматично.

6. Запуск проєкту
  1.	У верхній панелі Visual Studio обрати профіль IIS Express або https
  2.	Натиснути кнопку Run або клавішу F5
  3.	Після запуску у браузері відкриється вебзастосунок
